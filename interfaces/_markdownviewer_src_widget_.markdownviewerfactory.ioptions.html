Optional defaultFor

defaultFor: string[]

Inherited from IWidgetFactoryOptions.defaultFor

  • Defined in docregistry/src/registry.ts:798

The file extensions for which the factory should be the default.

Notes

Use "*" to denote all files. Specific file extensions must be preceded with '.', like '.png', '.txt', etc. Entries in this attribute must also be included in the fileExtensions attribute. The default is an empty array.

See also: fileExtensions.
