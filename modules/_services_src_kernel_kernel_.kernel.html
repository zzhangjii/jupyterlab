connectTo

  • connectTo(id: string, settings?: ISettings): Promise<IKernel>
    • Defined in services/src/kernel/kernel.ts:499

    Connect to a running kernel.

    Parameters

    • id: string

      The id of the running kernel.

    • Optional settings: ISettings

      The server settings for the request.

    Returns Promise<IKernel>

    A promise that resolves with the kernel object.

    Notes

    If the kernel was already started via startNewKernel, the existing Kernel object info is used to create another instance.

    Otherwise, if options are given, we attempt to connect to the existing kernel found by calling listRunningKernels. The promise is fulfilled when the kernel is running on the server, otherwise the promise is rejected.

    If the kernel was not already started and no options are given, the promise is rejected.

startNew

  • startNew(options?: IOptions): Promise<IKernel>
    • Defined in services/src/kernel/kernel.ts:472

    Start a new kernel.

    Parameters

    • Optional options: IOptions

      The options used to create the kernel.

    Returns Promise<IKernel>

    A promise that resolves with a kernel object.

    Notes

    Uses the Jupyter Notebook API and validates the response model.

    If no options are given or the kernel name is not given, the default kernel will by started by the server.
